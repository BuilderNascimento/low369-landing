# LOW369 Landing Page

Landing page profissional para captura de parceiros e influenciadores.

## 🚀 Tecnologias

- Next.js 15
- TypeScript
- Tailwind CSS
- Shadcn/ui

## 📦 Instalação

```bash
npm install
npm run dev
```

## 🌐 Deploy

Deploy automático via Vercel conectado ao GitHub.

## ✨ Funcionalidades

- Design responsivo
- Animações suaves
- Formulário de cadastro
- Carrossel de benefícios
- Seções interativas
